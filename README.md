# Hospital Management System
A web-based hospital management system built with PHP, HTML, CSS and SQL.

## Features
- Patient and staff account management
- Appointment scheduling and record tracking
- Medical history management
- Role-based access control (admin vs staff)
- Secure login/logout system

## Tech Stack
PHP, HTML, CSS, JavaScript, SQLite

## Grade: 2:1
Built as part of first year university coursework at Sheffield Hallam University.
